# Personal CV & Cybersecurity Portfolio Site

A personal CV/portfolio website built in PHP, showcasing my background in data science and cloud computing, alongside interactive cybersecurity tools built for coursework.

🔗 **Live Demo:** [View the live site](https://acbf6e5b-6949-42c3-8ad7-64a822bc4d0a-00-3uuhqxkda42ul.spock.replit.dev/)

> Note: This is a Replit development preview link and may not be active if the Repl is asleep. For guaranteed access, please view the source code directly in this repository.

## About

This site was built as part of an Information and Cyber Security course, combining a professional CV/resume presentation with hands-on cybersecurity utilities (hashing and encryption tools) to demonstrate practical implementation of security concepts.

## Pages

| Page | File | Description |
|---|---|---|
| Home | `index.php` | CV/resume homepage — objective, experience, education, skills, certifications, and academic projects |
| Cybersecurity Info | `CyberInfor.php` | Overview of cybersecurity concepts |
| Hashing | `cyberApplied1.php` | Interactive SHA-512 hash generator |
| Encryption | `cyberApplied2.php` | Interactive text encryption/decryption tool |
| Course Reflection | `courseReflectKeerthi.php` | Reflection on course learnings |
| Contact Us | `contactUsKeerthi.php` | Contact form |
| Registration | `regKeerthi.php` | Registration form |
